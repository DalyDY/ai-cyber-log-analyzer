# AI Cyber Log Analyzer

AI Cyber Log Analyzer is a machine learning cybersecurity project that detects suspicious network traffic from log data.

## Project Goal

The goal of this project is to build a simple AI powered intrusion detection tool that can analyze network traffic data, classify activity as benign or suspicious, and generate useful report.

## Planned Features

- Load and clean cybersecurity network log data
- Train a machine learning model to detect suspicious traffic
- Predict whether new traffic is benign or malicious
- Generate a basic security summary report
- Optional: build a Streamlit dashboard

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
- Streamlit
